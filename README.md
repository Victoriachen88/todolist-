# 待办清单 (ToDo List)

一个极简、清爽的网页版待办清单应用，灵感来自 MarkTodo。单文件、零依赖、纯前端，数据保存在浏览器本地。

## ✨ 功能特性

- **分类管理**：创建多个列表（如「A类」「投简历」），每个列表下挂多条任务
- **任务操作**：勾选完成、收藏星标（☆/★）、删除
- **多视图切换**：
  - 全部列表：所有分类卡片网格展示
  - 单个分类：聚焦查看某一类
  - 星标：聚合所有已收藏任务
  - 计划：按「进行中 / 已完成」分组
- **本地持久化**：所有数据通过 `localStorage` 自动保存，刷新/关闭浏览器不丢失
- **响应式布局**：桌面端左侧固定侧边栏 + 右侧主内容区；移动端自动收成抽屉式导航
- **零依赖**：单个 HTML 文件，内联 CSS + JavaScript，无需任何框架或图标库

## 🚀 使用方法

直接双击 `todo.html` 用浏览器打开即可使用，无需安装任何环境。


## 📁 文件结构

```
todolist/
├── todo.html     # 主程序（单文件，含全部 HTML/CSS/JS）
└── README.md     # 本说明文档
```

## 🛠 技术栈

- 原生 HTML5 + CSS3 + JavaScript（ES6）
- 数据存储：`localStorage`
- 无第三方库、无构建步骤

## 📝 数据说明

- 数据仅保存在**当前浏览器**的 `localStorage` 中（键名 `todo_categories`）
- 清除浏览器数据 / 更换设备会导致数据丢失，重要数据请及时备份
- 在「设置」页面可一键清空所有数据
